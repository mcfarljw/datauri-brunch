## datauri-brunch
[![Build Status](https://travis-ci.org/mcfarljw/datauri-brunch.svg?branch=master)](https://travis-ci.org/mcfarljw/datauri-brunch)

## Usage
Install the plugin via npm with `npm install --save-dev datauri-brunch`.

### Configuration

```javascript
module.exports = {
    plugins: {
        datauri: {
            pattern: /\.(gif|jpg|png)/
        }
    }
};
```
